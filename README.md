# 🌦️ WeatherSense Pro

A modern weather forecast web application built using **HTML, CSS, JavaScript, and the OpenWeatherMap API**. The application provides real-time weather information for any city with a clean and responsive user interface.

## 🚀 Features

- Real-time weather data
- Search weather by city name
- Current temperature
- Weather condition
- Humidity
- Wind speed
- Weather icons
- Responsive design
- User-friendly interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## 📂 Project Structure

```
WeatherSense-Pro
│── index.html
│── style.css
│── script.js
│── images
│── README.md
```

## 🔑 API

This project uses the **OpenWeatherMap API** to fetch live weather information.
